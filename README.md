 Trainer.py用来生成PinYin.py运行必须要用的Freq_Tree.json文件。
 因为在Mac里调用os库对文件进行删除操作，在Windows上运行的时候很容易出错，
 所以假如运行的话，可能会发现产生了一堆临时文件。这些文件会被定期清空，但没有删除。
 可以手动删，只要保留Freq_Tree.json就可以了。
